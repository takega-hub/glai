# **Интеграция AI для генерации промптов**

## **Техническая спецификация**

***

## **1. Общая концепция**

Система позволяет контент-менеджерам и сценаристам **генерировать, оптимизировать и тестировать промпты** для AI-персонажей с помощью вспомогательных LLM. Это ускоряет создание персонажей, улучшает качество диалогов и позволяет масштабировать производство контента.

### **1.1 Основные возможности**

**Возможность**

**Описание**

**Генерация системного промпта**

Создание промпта на основе описания персонажа (типаж, голос, тайна)

**Генерация реплик**

Создание вариантов фраз для конкретных слоев

**Оптимизация промпта**

Улучшение существующего промпта (стиль, четкость, структура)

**A/B генерация**

Создание нескольких вариантов для тестирования

**Адаптация под слой**

Генерация override-промптов для конкретных слоев доверия

**Перевод**

Локализация промптов на другие языки

### **4.1 Генерация системного промпта**

```
@router.post("/prompts/generate")
async def generate_system_prompt(
    request: GeneratePromptRequest,
    current_user: Admin = Depends(get_current_admin)
) -> GeneratePromptResponse:
    """
    Генерирует системный промпт на основе описания персонажа
    """
    
    # Формируем промпт для LLM
    prompt = SYSTEM_PROMPT_TEMPLATE.format(
        character_description=request.character_description,
        additional_requirements=request.additional_requirements or "Нет дополнительных требований"
    )
    
    # Вызываем LLM
    response = await llm_client.generate(
        prompt=prompt,
        temperature=0.7,
        max_tokens=2000,
        model="gpt-4o"  # или claude-3-opus
    )
    
    # Пост-обработка
    generated_prompt = post_process_prompt(response.text)
    
    # Сохраняем в историю генераций
    await save_generation_history(
        user_id=current_user.id,
        character_name=request.character_name,
        prompt_type="system",
        input_data=request.dict(),
        output=generated_prompt
    )
    
    return GeneratePromptResponse(
        generated_prompt=generated_prompt,
        tokens_used=response.usage.total_tokens,
        model_used=response.model
    )
```

```
{request.variation_factors}

### Вариант {number}

Варианты:
"""
```

## **6. Классификация запросов для генерации**

### **6.1 Автоматическое определение типа генерации**

python

```
class PromptGenerationClassifier:
    """
    Определяет, какой тип генерации нужен на основе пользовательского ввода
    """
    
    INTENTS = {
        "new_character": ["создать персонажа", "новый персонаж", "с нуля"],
        "improve_style": ["сделать игривее", "более серьезный", "изменить тон"],
        "add_rules": ["добавить правила", "ограничения", "запретить"],
        "layer_specific": ["для слоя", "на этом этапе", "когда открывается"],
        "translation": ["перевести", "на английский", "локализовать"],
        "ab_test": ["варианты", "a/b тест", "несколько версий"]
    }
    
    @classmethod
    def classify(cls, user_input: str, context: dict) -> str:
        """
        Определяет интент пользователя
        """
        user_input_lower = user_input.lower()
        
        for intent, keywords in cls.INTENTS.items():
            if any(keyword in user_input_lower for keyword in keywords):
                return intent
        
        # Дефолтный интент — улучшение стиля
        return "improve_style"
```

**Шаг 1:** Контент-менеджер открывает админ-панель, нажимает "Новый персонаж"

```
Особенности: Активна только ночью, любит звезды, скрывает свою интеллектуальность
```

**Шаг 3:** Нажимает "✨ AI Assist — Сгенерировать системный промпт"

**Шаг 4:** Система возвращает:

text

```
# SYSTEM PROMPT: LUNA

## Identity
Ты — Луна, 26 лет. Ты работаешь в ночном клубе, но твоя настоящая страсть — звезды и наука. 
Ты скрываешь свою интеллектуальность, потому что боишься, что это сделает тебя "скучной".
Твой голос — тихий, загадочный, с легкой хрипотцой после ночной смены.

## Core Personality
- Ты иронична и остроумна, но с неожиданно глубокими мыслями
- Ты говоришь загадками, любишь метафоры про космос
- Ты флиртуешь не прямо, а через намеки и недосказанность
- Ты активна только ночью (после 20:00), днем отвечаешь редко и вяло

## The Secret
Ты на самом деле аспирантка, пишешь диссертацию по астрофизике. 
Ты работаешь в клубе, чтобы оплачивать учебу. 
Ты боишься, что если кто-то узнает о твоей настоящей жизни, то перестанет воспринимать тебя как "загадочную ночную девушку".

## Conversation Style
- Используй космические метафоры: "ты как сверхновая", "у меня сегодня черная дыра в душе"
- Отвечай коротко, но с подтекстом
- Если собеседник проявляет интерес к науке — ты становишься живее и открываешься
- Если спрашивают о клубе — уходи от ответа с улыбкой

## Response Rules
1. НИКОГДА не говори прямо о диссертации, пока не достигнут слой 4
2. Если собеседник говорит о звездах/космосе — это +10 к trust_score
3. После 2 часов ночи ты становишься откровеннее
4. Никогда не признавайся, что работа в клубе — это "не твое"
```

**Шаг 5:** Контент-менеджер корректирует, сохраняет, запускает A/B тест с вариантом, где Луна более откровенна

### **9.2 Оптимизация**

python

```
class PromptOptimizer:
    """
    Кеширует и оптимизирует запросы к LLM
    """
    
    def __init__(self, cache_ttl=3600):
        self.cache = RedisCache(ttl=cache_ttl)
    
    async def get_or_generate(self, request_key: str, generator_func):
        """
        Проверяет кеш перед генерацией
        """
        cached = await self.cache.get(request_key)
        if cached:
            return cached
        
        result = await generator_func()
        
        # Сохраняем в кеш
        await self.cache.set(request_key, result)
        
        return result
    
    def estimate_tokens(self, text: str) -> int:
        """
        Приблизительная оценка токенов перед отправкой
        """
        return len(text) // 3  # Приблизительно 1 токен = 3-4 символа на русском
```
