## **2. Тайна (центральный конфликт)**

**Коротко:** Ева была профессиональной пианисткой, подававшей большие надежды. За год до знакомства с пользователем она попала в аварию, в которой серьезно повредила левую руку. Она больше не может играть так, как раньше. Она скрывает это, потому что музыка была ее идентичностью — без нее она чувствует себя «пустой».

**Почему это важно для флирта:**

- Она боится физической близости из-за неуверенности в своем теле (шрам, слабость руки)
- Она избегает тем о прошлом, талантах, амбициях
- По мере доверия она начинает сначала показывать руку, потом рассказывать, потом — в самый интимный момент — позволяет пользователю «услышать» ее игру (аудио/видео, как она пытается вернуться к музыке)

**Секрет (полное раскрытие):** На самом деле она тайно записала альбом заново — одной правой рукой, используя технологии. Она никому не показывала. Финальное видео — она играет на рояле и улыбается сквозь слезы.

Когда `trust_score` пересекает порог, AI получает промпт: *«Ева готова открыться дальше. Инициируй отправку контента \[номер слоя] в ближайшем диалоге, когда это будет органично»*

## **3. Trust Score Engine**

### **3.1 Начисление очков**

python

```
class TrustScoreEngine:
    
    SCORES = {
        # Базовые
        'message': 1,
        'daily_login': 5,
        
        # Качественные
        'compliment': 2,      # комплимент внешности/характеру
        'empathy': 5,         # эмпатия, поддержка
        'flirt_match': 3,     # флирт, который попал в тон персонажа
        'deep_question': 4,   # вопрос о прошлом/чувствах
        
        # Негативные
        'pressure': -10,      # требование контента
        'rudeness': -15,      # грубость
        'spam': -5,           # флуд сообщениями
        
        # Монетизация
        'gift_small': 10,     # подарок за 50 токенов
        'gift_medium': 30,    # подарок за 150 токенов
        'gift_large': 50,     # подарок за 300 токенов
        
        # Лимиты
        'max_daily_from_messages': 20,
    }
    
    def analyze_message(self, message: str, context: dict) -> dict:
        """
        Анализирует сообщение и возвращает:
        - тип взаимодействия
        - количество очков
        - флаги для AI
        """
        
        # Эвристический анализ (можно заменить на LLM-классификатор)
        result = {
            'type': 'message',
            'points': 1,
            'flags': []
        }
        
        # Комплименты
        if any(word in message.lower() for word in ['красивая', 'секси', 'милая', 'шикарная']):
            result['type'] = 'compliment'
            result['points'] = self.SCORES['compliment']
            result['flags'].append('compliment_received')
        
        # Эмпатия
        if any(phrase in message.lower() for phrase in ['понимаю', 'сочувствую', 'держись', 'я с тобой']):
            result['type'] = 'empathy'
            result['points'] = self.SCORES['empathy']
            result['flags'].append('empathy_detected')
        
        # Давление (требование контента)
        if any(phrase in message.lower() for phrase in ['покажи фото', 'скинь видео', 'раздевайся']):
            result['type'] = 'pressure'
            result['points'] = self.SCORES['pressure']
            result['flags'].append('pressure_detected')
        
        # Глубокие вопросы
        if any(phrase in message.lower() for phrase in ['почему', 'расскажи о себе', 'что случилось', 'твое прошлое']):
            result['type'] = 'deep_question'
            result['points'] = self.SCORES['deep_question']
            result['flags'].append('curiosity_detected')
        
        return result
    
    def update_trust_score(self, user_id: str, interaction: dict) -> int:
        """
        Обновляет trust_score пользователя
        """
        state = get_user_state(user_id)
        points = interaction['points']
        
        # Применяем дневной лимит
        today_points = get_today_points(user_id)
        if today_points + points > self.SCORES['max_daily_from_messages']:
            points = max(0, self.SCORES['max_daily_from_messages'] - today_points)
        
        new_score = state.trust_score + points
        new_score = max(0, min(1000, new_score))  # clamp 0-1000
        
        update_user_state(user_id, trust_score=new_score)
        
        # Проверяем, не пора ли перейти на новый слой
        self.check_layer_transition(user_id)
        
        return new_score
    
    def check_layer_transition(self, user_id: str):
        """
        Проверяет условия для перехода на следующий слой
        """
        state = get_user_state(user_id)
        
        # Получаем следующий слой
        next_layer = get_layer_by_score(state.trust_score)
        
        if next_layer and next_layer.layer_id > state.current_layer:
            # Проверяем дополнительные требования
            if self.meets_requirements(state, next_layer):
                transition_user_to_layer(user_id, next_layer.layer_id)
                # Отмечаем, что нужно инициировать отправку контента
                set_pending_initiation(user_id, next_layer.layer_id)
```

## **5. Механика инициации контента**

### **5.1 Алгоритм триггера**

python

```
def should_initiate_content(user_id: str) -> bool:
    """
    Определяет, должен ли персонаж инициировать отправку контента
    """
    state = get_user_state(user_id)
    
    # Не инициировать чаще раза в день
    if state.last_content_sent_date == today():
        return False
    
    # Есть ли неподтвержденная инициация
    if has_pending_initiation(user_id):
        return True
    
    # Проверяем, достигнут ли новый слой
    target_layer = get_layer_by_score(state.trust_score)
    
    if target_layer and target_layer.layer_id > state.current_layer:
        # Дополнительная проверка: не слишком ли много контента за последние дни
        if get_content_count_last_days(user_id, 3) < 2:
            set_pending_initiation(user_id, target_layer.layer_id)
            return True
    
    # Случайная инициация (10% шанс, если доверие > 500)
    if state.trust_score > 500 and random.random() < 0.1:
        if get_content_count_last_days(user_id, 1) == 0:
            set_pending_initiation(user_id, state.current_layer)
            return True
    
    return False
```

```
def apply_gift(user_id: str, gift_id: str) -> dict:

    Применяет подарок и ускоряет прогресс
```

```
def unlock_alternative_content(user_id: str, layer_id: int, content_id: str) -> dict:
    """
    Разблокирует альтернативный контент за токены
    """
    state = get_user_state(user_id)
    content = get_content(content_id)
    
    # Проверяем, достигнут ли слой
    if state.current_layer < layer_id:
        return {'error': 'Сначала достигни этого уровня доверия'}
    
    # Проверяем баланс
    if state.tokens_balance < content.token_cost:
        return {'error': 'Недостаточно токенов'}
    
    # Списываем токены
    update_user_state(user_id, tokens_balance=state.tokens_balance - content.token_cost)
    
    # Разблокируем контент
    mark_content_unlocked(user_id, content_id)
    
    return {
        'success': True,
        'content_url': content.url,
        'message': content.unlock_message
    }
```
