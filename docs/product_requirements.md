# Требования к продукту: AI Chat с персонажами

## Обзор продукта
Приложение для взаимодействия пользователей с AI-персонажами через чат, с возможностью отправки фото, подарков и разблокировки контента.

## Основные функции

### 1. Система персонажей
- Создание и управление AI-персонажами
- Настройка внешности, личности и поведения
- Система уровней доверия (layers)
- Уникальные фотографии и контент для каждого персонажа

### 2. Чат-система
- Реал-тайм общение с AI-персонажами
- Отправка и получение сообщений
- Система многокомпонентных ответов (message_parts)
- Анимация печати с динамическим временем

### 3. Система фото
- **Запрос фото от пользователя**: Пользователь может запросить фото, AI анализирует запрос и выбирает подходящее фото из галереи персонажа
- **Автоматическая отправка фото**: Персонаж может самостоятельно присылать фото из своей коллекции
- **Генерация новых фото**: Возможность генерации интимных фото за подарки
- **Соответствие запросу**: AI должен анализировать описание запроса и подбирать фото по описанию из базы данных

### 4. Система подарков
- Три типа подарков: small, medium, large
- Увеличение уровня доверия при отправке подарков
- Разблокировка контента за подарки
- Система токенов для оплаты подарков

### 5. Монетизация
- Система токенов для оплаты функций
- Покупка подарков за токены
- Генерация фото за токены
- История транзакций

## Требования к AI-диалогу

### Анализ запроса на фото
Когда пользователь просит фото, AI должен:
1. Проанализировать текст запроса пользователя
2. Определить контекст и тип желаемого фото
3. Поискать подходящее фото в базе данных персонажа
4. Если найдено подходящее фото - отправить его
5. Если не найдено - предложить генерацию нового фото за подарок

### Примеры запросов и реакций:
- "Скинь фотку" → Поиск случайного фото из доступных
- "Покажи себя в красном платье" → Поиск фото с красным платьем в описании
- "Хочу тебя увидеть" → Поиск интимного фото (если доступно)
- "Сделай фото специально для меня" → Предложение генерации за подарок

### Критерии выбора фото:
- Соответствие описанию запроса
- Уровень доверия пользователя
- Доступность фото (не заблокировано)
- Контекст предыдущего общения

## Технические требования

### База данных
- Таблицы: characters, users, messages, photos, gifts, transactions
- Связи между персонажами, пользователями и контентом
- Система блокировки контента по уровням

### API Endpoints
- `/dialogue/send-message` - отправка сообщений
- `/dialogue/history/{character_id}` - история чата
- `/dialogue/send-gift` - отправка подарков
- `/dialogue/generate-on-demand-photo` - генерация фото за токены

### Безопасность
- Аутентификация пользователей
- Проверка прав доступа к контенту
- Валидация входных данных
- Защита от злоупотреблений

## Критерии успеха
- Пользователи могут естественно запрашивать фото
- AI корректно интерпретирует запросы и подбирает релевантные фото
- Система подарков работает плавно и интуитивно
- Чат выглядит естественно с динамической анимацией печати
- Все фото и контент корректно сохраняются и отображаются в истории
