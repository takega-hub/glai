# Техническая архитектура: AI Chat с персонажами

## Общая архитектура

### Стек технологий
- **Frontend**: React + TypeScript + Vite + Tailwind CSS
- **Backend**: Python + FastAPI + PostgreSQL
- **AI Engine**: OpenAI GPT для диалогов, ComfyUI для генерации изображений
- **База данных**: PostgreSQL с Supabase
- **Аутентификация**: JWT токены
- **Файловое хранилище**: Локальное хранилище для изображений

## Компоненты системы

### 1. Frontend (React)

#### Основные компоненты:
- **UserChat**: Основной компонент чата с персонажем
- **CharacterCard**: Карточка персонажа в списке
- **GiftModal**: Модальное окно для отправки подарков
- **TypingIndicator**: Анимация печати
- **PhotoGallery**: Галерея фотографий персонажа

#### Состояние приложения:
- **authStore**: Аутентификация пользователя
- **characterStore**: Состояние персонажей
- **chatStore**: Состояние чатов

### 2. Backend (FastAPI)

#### Основные маршруты:
- **/dialogue/**: Управление диалогами
  - `POST /send-message`: Отправка сообщения
  - `GET /history/{character_id}`: Получение истории чата
  - `POST /send-gift`: Отправка подарка
  - `POST /generate-on-demand-photo`: Генерация фото

#### Сервисы:
- **AIDialogueEngine**: Генерация AI-ответов
- **TrustScoreEngine**: Управление уровнем доверия
- **PhotoService**: Управление фотографиями
- **GiftService**: Управление подарками

### 3. Система фото

#### Типы фото:
1. **Существующие фото**: Из галереи персонажа
2. **Сгенерированные фото**: Созданные через ComfyUI
3. **Тизеры**: Базовые фото для начального уровня
4. **Эксклюзивные фото**: За подарки и высокий уровень доверия

#### Логика выбора фото:
```python
# Псевдокод для AI анализа запроса на фото
def analyze_photo_request(user_request, character_photos, trust_level):
    # 1. Парсинг запроса пользователя
    keywords = extract_keywords(user_request)
    intent = classify_intent(user_request)  # casual, intimate, specific
    
    # 2. Поиск подходящих фото
    matching_photos = search_photos_by_description(character_photos, keywords)
    
    # 3. Фильтрация по уровню доверия
    available_photos = filter_by_trust_level(matching_photos, trust_level)
    
    # 4. Принятие решения
    if available_photos:
        return select_best_photo(available_photos, user_request)
    elif intent == "intimate" and trust_level >= required_level:
        return propose_gift_for_exclusive_photo()
    else:
        return propose_photo_generation()
```

### 4. Система AI-диалога

#### Компоненты:
- **GPT Engine**: Генерация текстовых ответов
- **Prompt Builder**: Создание контекстных промптов
- **Response Parser**: Парсинг JSON-ответов с действиями
- **Context Manager**: Управление контекстом разговора

#### Типы ответов:
1. **Текстовые сообщения**: Обычные текстовые ответы
2. **Многокомпонентные сообщения**: Ответы с message_parts для анимации
3. **Действия**: Специальные действия (фото, подарки и т.д.)

#### Пример JSON-ответа:
```json
{
  "response": "Текстовый ответ персонажа",
  "message_parts": ["Часть 1", "Часть 2", "Часть 3"],
  "action": {
    "type": "send_photo",
    "photo_id": "uuid",
    "description": "Описание фото"
  },
  "trust_score_change": 5
}
```

## Алгоритмы

### 1. Выбор фото по запросу
1. **Анализ запроса**: Извлечение ключевых слов и намерений
2. **Поиск по описанию**: Сопоставление с описаниями фото
3. **Фильтрация**: Проверка доступности по уровню доверия
4. **Ранжирование**: Сортировка по релевантности
5. **Выбор**: Отправка лучшего совпадения

### 2. Генерация ответа AI
1. **Сбор контекста**: История чата, уровень доверия, профиль персонажа
2. **Создание промпта**: Формирование контекстного промпта для GPT
3. **Генерация ответа**: Получение ответа от GPT
4. **Парсинг действий**: Извлечение действий из JSON
5. **Сохранение**: Сохранение ответа в базу данных

### 3. Управление уровнем доверия
1. **Анализ сообщения**: Определение типа и содержания
2. **Расчет изменения**: Вычисление очков доверия
3. **Проверка порогов**: Проверка достижения новых уровней
4. **Разблокировка контента**: Открытие нового контента
5. **Уведомление**: Информирование пользователя
