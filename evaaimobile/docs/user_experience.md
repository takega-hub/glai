# **Пользовательский интерфейс и опыт (User Experience)**

Этот документ описывает пользовательский интерфейс и опыт для основной,面向 пользователя части платформы.

***

## **📱 ЭКРАН 1: ПАНЕЛЬ УПРАВЛЕНИЯ ПОЛЬЗОВАТЕЛЯ (ДАШБОРД)**

### **После входа в систему пользователь попадает на главный экран, где он может:**

- **Видеть своих персонажей:** Список персонажей, с которыми он взаимодействует.
- **Продолжить общение:** Быстрый доступ к последнему активному чату.
- **Управлять своим профилем:** Доступ к настройкам профиля.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  👤 [Аватар] Алексей                                    ⚙️ Настройки │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                                                              ││
│  │  **Ваши персонажи**                                          ││
│  │                                                              ││
│  │  ┌─────────┐    ┌─────────┐                                  ││
│  │  │         │    │         │                                  ││
│  │  │   Ева   │    │   Луна  │                                  ││
│  │  │ [Продолжить]│    │ [Продолжить]│                                  ││
│  │  └─────────┘    └─────────┘                                  ││
│  │                                                              ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## **💬 ЭКРАН 2: ИНТЕРФЕЙС ЧАТА**

### **Основной экран для взаимодействия с персонажем:**

- **Диалоговое окно:** Отображение истории сообщений.
- **Поле ввода:** Для отправки сообщений персонажу.
- **Индикатор доверия:** Визуальное отображение уровня "доверия" с персонажем.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  ←  Чат с Евой                                    доверие: 75%   │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                                                              ││
│  │  Ева: Привет! Рада снова тебя видеть.                        ││
│  │                                                              ││
│  │  Вы: И я тебя.                                               ││
│  │                                                              ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │ [Сообщение...                                    ] [➤]      ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## **📈 ЭКРАН 3: СИСТЕМА ПРОГРЕССИИ И РАЗБЛОКИРОВКИ**

### **Как пользователь будет видеть свой прогресс и разблокированный контент:**

- **Уведомления:** Сообщения о достижении нового "уровня доверия".
- **Галерея контента:** Раздел, где пользователь может просматривать разблокированные фото и видео.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  🖼️ Галерея Евы                                        ← Назад      │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                                                              ││
│  │  **Разблокированный контент**                                ││
│  │                                                              ││
│  │  ┌─────────┐ ┌─────────┐ ┌─────────┐                         ││
│  │  │ [Фото 1]│ │ [Видео 1]│ │ [Фото 2]│                         ││
│  │  └─────────┘ └─────────┘ └─────────┘                         ││
│  │                                                              ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## **👤 ЭКРАН 4: ПРОФИЛЬ ПОЛЬЗОВАТЕЛЯ**

### **Пользователи могут просматривать и редактировать свой профиль:**

- **Персональная информация:** Имя, аватар, и т.д.
- **Статистика:** Общее время в чатах, количество разблокированного контента.

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                  │
│  👤 Профиль                                            ← Назад      │
│                                                                  │
│  ┌─────────────────────────────────────────────────────────────┐│
│  │                                                              ││
│  │  [Аватар]  **Алексей**                                       ││
│  │                                                              ││
│  │  **Статистика:**                                             ││
│  │  - Время в чатах: 15 часов                                   ││
│  │  - Разблокировано: 12 фото, 3 видео                          ││
│  │                                                              ││
│  └─────────────────────────────────────────────────────────────┘│
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```
